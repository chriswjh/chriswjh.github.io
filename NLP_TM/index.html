Natural Language Processing and Text Mining, Spring 2026 (Undergraduates)

This course introduces the basic concepts in natural language processing and the applications in text mining. This is offered as an elective course in the undergraduate program.

Homework Assignments

Please hand in your homework in paper form or in electronic files before deadline.
Programs or homework in electronic files must be submitted to me via our submission site as follows:
  • Program Submission Site: iSchool+
  • User Account: your account in Taipei Tech Portal
  • Program Uploading: Please compress source codes and documentation in a .zip file, and clearly name your program files.
    Note: If your program needs special environment or tool to compile or run, please write documentation about it.
