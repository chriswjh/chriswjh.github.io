Operating Systems, Spring 2021 (Undergraduates)

This course offers an introduction to the concepts and principles in modern operating systems. This is offered as a required course in the undergraduate program.

  • TA: (TBD) (R1424, Technology Building)
  • Textbook: Operating System Concepts, 9th ed., by Abraham Silberschatz, Peter Baer Galvin, and Greg Gagne, John Wiley & Sons, Inc, 2013.
    • Note: The latest edition is the 10th edition, which is available as "Asia Edition" in Taiwan. I will try to cover some of the latest topics whenever possible.
    • Note: Earlier editions (such as 8th ed. or Java version) of the textbook is also acceptable. Changes in the latest edition will be mentioned whenever applicable.
    • Note: Please do not confuse it with the authors' new version called "Operating System Concepts Essentials, 2nd Edition", which covers most of the core topics in the contemporary operating systems. Some chapters were removed, such as Deadlocks, Virtual Machines, and Distributed Systems, some of which will still be covered in this course.
  • References:
    • Modern Operating Systems, 4th ed., by Andrew S. Tanenbaum and Herbert Bos, Prentice-Hall, 2015. (MOS4)
